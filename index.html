Inspired by speed reading apps like Sprtiz, I have created a small tool for increasing reading time. Simply paste the text you want to read into the input box, along with the desired words per minute. The average reading speed is around 250 wpm, while top speed readers boast a reading speed of 1000 wpm with about 50% comprehension. Read more about speed reading here.
Typing test:
To use, start typing the text above the input box. Try to get as many words correctly spelled in 60s. The words per minutte will be displayed to the right after.
This is a plot using the d3.js library. It plots the surface area of a cylinder (in3) against the radius (inches) for a fixed volume. The blue line is for 12 oz where the surface area is optimal at 1.5 in. The red line is 7.5 oz, or the same volume as the smaller cans of soda.
